About Parts of Speech Tagger

Part-of-speech tagging is the process of associating each word in a text with it’s part-of-speech category and possibly a set of morphosyntactic features.

The basic idea is to apply a set of rules to an initial annotation achieved using a simple algorithm.

We also addresses the problem of tagging unknown words, i.e. words that don’t appear in the training corpus.

Features

Eric Brill method is used.

An ‘error-driven’ approach for learning an ordered set of rules on Transformation-Based Learning by Christian Siefkes.

Hybrid model of both rule-based and ML techniques for accuracy POS tagging.

Open source build and redistribute under MIT License.
